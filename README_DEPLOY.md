# Application mobile – Purification Taq-His8

Fonctionnement :
- écran mobile optimisé ;
- bouton noir GO ;
- chrono de 15 secondes ;
- sélection libre des étapes ;
- verrouillage après 15 secondes ;
- aucune indication de bonne ou mauvaise réponse.

Pour l'utiliser avec un jury :
1. Héberger `index.html` sur un hébergement statique : GitHub Pages, Netlify, serveur UCBL, Moodle, etc.
2. Copier l'URL publique obtenue.
3. Remplacer le QR code de la slide par un QR généré avec cette URL.

Le QR dans la présentation fournie est provisoire, car une page locale ChatGPT n'est pas accessible aux téléphones du jury.
